# Task Manager Backend: Task Management Overview

## Purpose

Manages task CRUD operations, filtering, and search for authenticated users.

## Responsibilities

- Task creation, viewing, updating, and deletion
- Task filtering by status and priority
- Task search by title and description

## Key Capabilities

- Create tasks with title, description, status, and priority
- List tasks with optional filtering and search
- View individual tasks
- Update task details
- Delete tasks
